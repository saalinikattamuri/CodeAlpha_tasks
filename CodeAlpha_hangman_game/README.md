# Hangman Game (Python)

This is a simple text-based Hangman game created using Python.
The player guesses the hidden word one letter at a time. The player has a limited number of incorrect guesses to find the word.

## Features

* Random word selection
* 6 incorrect guesses allowed
* Displays guessed letters
* Win or lose message

## How to Run

Run the program using:

```
python main.py
```

## Concepts Used

* Python
* Random module
* Loops
* If-else conditions
* Lists and strings
