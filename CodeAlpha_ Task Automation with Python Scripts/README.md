# Python Task Automation Tool

This project automates common tasks using Python.

Features:
- Move JPG files automatically
- Extract email addresses from text files
- Scrape webpage titles

Technologies Used:
- Python
- os module
- shutil module
- Regular Expressions (re)
- requests library
- File handling

How to Run:
1. Install requests
2. Run: python automation_tool.py
3. Choose the task from the menu
